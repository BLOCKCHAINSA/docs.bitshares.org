Account History API¶

class graphene::app::history_api¶

The history_api class implements the RPC API for account history.

This API contains methods to access account histories

Public Functions

vector<operation_history_object> get_account_history(account_id_type account, operation_history_id_type stop, unsigned limit, operation_history_id_type start) const¶

Get operations relevant to the specificed account.

Return
A list of operations performed by account, ordered from most recent to oldest.
Parameters
  • account -

    The account whose history should be queried

  • stop -

    ID of the earliest operation to retrieve

  • limit -

    Maximum number of operations to retrieve (must not exceed 100)

  • start -

    ID of the most recent operation to retrieve
