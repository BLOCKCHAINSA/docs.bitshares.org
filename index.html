Welcome to the Graphene Documentation¶

BitShares 2.0 is a Financial Smart Contracts platform that is based on the Graphene technology built by Cryptonomex . You can see Graphene as a toolkit for real-time blockchains.

Below, you can find the Graphene documentation. Since BitShares 2.0 is the first application of Graphene technology, you will be able to use almost everything in BitShares 2.0, too. We separated the documentation into smaller parts for convenience and for the sake of easy location of relevant information.
